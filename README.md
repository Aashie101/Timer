# Timer
# 🌸 Cute Study Timer

A simple and beginner-friendly countdown timer built with Python and Tkinter. The application features a cute pink-themed graphical user interface with Start, Pause, and Reset functionality, making it perfect for study sessions, Pomodoro practice, or general time management.

## Features

* Countdown timer in minutes and seconds format
* Start timer
* Pause timer
* Reset timer
* Cute pink-themed interface
* Built using Python's built-in Tkinter library
* Great beginner GUI project

## Technologies Used

* Python 3
* Tkinter

## How to Run

1. Make sure Python 3 is installed.
2. Save the source code as `timer.py`.
3. Open a terminal in the project folder.
4. Run:
python timer.py

## Usage

1. Enter the desired countdown time in seconds.
2. Click **Start** to begin the countdown.
3. Click **Pause** to temporarily stop the timer.
4. Click **Start** again to resume.
5. Click **Reset** to clear the timer and start over.

## Learning Concepts Demonstrated

* Functions
* Global variables
* Conditional statements (`if`, `elif`, `else`)
* Event-driven programming
* Tkinter widgets (`Label`, `Button`, `Entry`)
* GUI layout using `pack()`
* Timer scheduling with `window.after()`
